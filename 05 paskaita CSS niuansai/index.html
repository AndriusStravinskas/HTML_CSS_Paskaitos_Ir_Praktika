Tėviniai/Vaikiniai elementai

Vaikiniai elementai - Tai elementas esantis kažkokio elemento viduje yra vaikinis elementas, tam elementui, kurio viduje jis yra.
Tėviniai elementai - Tai elementas turintis viduje kitą elementą, yra tėvinis elementas, tam elementui, kuris yra jo viduje.

  • ul vaikas 2

    3. ol vaikas 3

Paveldėjimas

Paveldėjimas - - iš tėvinio elemento, vaikiniai elementai paveldi tam tikrus stilius. Paveldimi stiliai dažniausiai būna kažkas paprasto kaip - spalva, šriftas, dydis ir pns... Nepaveldimi yra dydžio, display, position, fono, rėmelių ir pns.. nustatymai.

Stilių prioritetai

  1. Inline Style - tai html įrašytas css style
  2. ID
  3. Class
  4. Element Selector
  5. CSS order (in CSS code) [CSS eiliškumas, skaito nuo viršaus į apačia/iš kairės į dešnę, todėl pritaikys paskutinį kurį perskaitė]
  6. Element default Style (pvz h1 default yra paryskintas t.t.)
  7. Inheritence (Paveldėjimas)

Esant dviem selektoriam, kurie nusako to paties elemento stilius yra atsižvelgiama į tų selektorių prioritetus. Aukščiau esantys selektoriai, turi didesnį prioritetą (žiūrėti į sąrašą).
Esant dviem vienodo prioriteto selektoriams (keičiant vienodą stilių) yra atsižvelgiama į CSS eiliškumą (kuris CSS parašytas vėliau, tas ir bus taikomas).

!important prierašas prie stiliaus ignoruoja visus prioritetus ir suteikia tą stilių ir niekas jo pakeisti negali apart kito !important, kuris turėtų aukštesnį prioritetą. jį galima parašyti CSS faile pvz.

background-image: linear-gradient(
to top right,
rgb(151, 161, 103),
rgb(100, 154, 161)
) !important;

Browser default style

Browser default style

Tag'ai gali turėti savitus (default) stilius, kurie jiem yra suteikiami pačių naršyklių
Kartais skirtingos naršyklės turi minimalistinius stilius. Tai gali paveikti svetainės išvaizda.
Pvz.: h1 ir h2 browseris padaro, jog h1 patampa tokiu pačiu dydžiu kaip ir h2.
Kartais numatytieji stiliai pagadia svetainės vaizdą arba apsunkina programuotojo darbą.

Yra keletas būdų išpresti nenorimų numatytų (default) stilių sukeliamas problemas.
Vienas iš būdų yra nuimti visus numatytuosius stilius nuo savo svetainėje naudojamų tag,ų: RESET CSS.
Kitas (plačiau naudojamas) būdas yra normalizuoti numatytuosius stilius tarp skirtingų naršyklių ir dalį jų panaikinti/pakeisti naudojant Normalize CSS.
