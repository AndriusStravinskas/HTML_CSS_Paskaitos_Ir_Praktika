Temos kurias mokinsimės

HTML

Hypertext markup language - hiper teksto žymėjimo kalba skirta surašyti informacijai į puslapį ir ją grupuoti tag'ais.


CSS

Cascading style sheets - stilizavimo kalba.


JS

Javascript - Skirtas pridėti puslapiui funkcionalumą ir/ar reagavimą į vartotojo įvestis.


JS DOM

Javascript Document Object Model - Javascript'u keičiamas/redaguojamas/pildomas puslapio kontentas.


TS

Typescript - Lygiai tas pats Javascript, tik su griežta rašymo struktūra, kurioje reikia nurodyti duomenų tipus.


React

React - Galingas svetainių kurimo variklis. Jo vienas didžiausių privalumų niekada nėra perkraunamas puslapis (jog nereikalautų laiko, kai perkraunami duomenys jog nesikreiptų į duomenų baze.).


NodeJS

Node Javascript - Skirtas kurti backend'ui, kuriame vykdomi įvairūs veiksmai su duomenimis.

Stilizavimo tag'ai

  • i - italic - itališkas srifto stilius, kuris pakreipia tekstą.
  • b - Bold - Paryškina tekstą.
  • em - Emphasize - pakreipia tekstą. (skiriasi nuo i tag'o jog screen rederiai pabrėžtų tekstą.)
  • strong - pastorina tekstą.(skiriasi nuo b tag'o jog screen rederiai pabrėžtų tekstą.)
  • sup - Super(man) - pakelia ir sumažina tekstą į viršų.
  • sub - SubScript - Sumažina tekstą ir nuleidžia į apačią.
  • mark - Pažymi tekstą geltonai.
  • u - Underline - Pabraukia tekstą.
  • small - Pamažina tekstą.
  • big
    • - Padidina tekstą.
  • center - Sucentruoja tekstinę informaciją (po table/img/paragraph).

Tag'ai

  • - Mygtuko tag'as
  • q - Short quotation - Trumpa citata.
  • blockquote - Long quotation - Ilga citata.
  • - Input - Formos įvesties laukelis.
  • label - Etiketė skirta nurodyti į nustatytą (pagal id) input'ą.
  • select - Pasirinkimo elementas, kurio viduje yra opcijos.
  • option - Select (Pasirinkimo) elemento viduje esanti opcija
  • textarea - Didelis tekstinis įvesties laukas.
  • script - skirtas rašyti/prijungti JS kodą/failą.
  • span - Tuščias linijinio tipo elementas skirtas išskirti informaciją.
  • fieldset - skirtas grupuoti susijusius input elementus formos viduje.
  • legend - Nurodo fieldset'o pavadinimą.
  • figure - skirtas img elementas grupuoti.
  • figcaption - aprašymui po nuotrauka.
  • pre - Kaip p tag'as jog reguotų kaip dėsi enter tekste.
  • video - Skirtas atvaizduoti kažkokį vaizdo įrašą.
  • iframe - skirtas į puslapį įkelti kito puslapio turinio arba dalį puslapio turinio (video/map/... su įvairiomis parinktimis).
  • table - Lentelės tag'as.
  • tr - Table Row - Lentelės eilutė.
  • th - Table Heading - Lentelės stulpelio/eilutės pavadinimas.
  • td - Table Data - Lentelės duomenys (info vieneto gabaliukas)
  • thead - Table Head - Lentelės galva (viršutinė dalis), kurioje būna stulpelių pavadinimai.
  • tbody - Table Body - Lentelės kūnas (vidurinė dalis), kurioje būna lentelės duomenys.
  • tfoot - Table Foot - Lentelės kojos ( apatinė dalis), kurioje būna išvada/suma/duomenys (gali būti skirta atkartoti thead).
  • colgroup - Column Group - Skirtas grupuoti stulpelius (stilizavimui).
  • caption - skirtas lentelės pavadinimui atvaizduoti.
