Box Model

  • Content - Plotis ir aukštis pvz.: teksto ar paveiksliuko, Pačiame centre.
  • Padding - Atitraukimas nuo contento iki rėmelio.(LT - pamušalas :) )
  • Border - Rėmeliai
  • Margin - Atitraukimas nuo elemento ribų iki sekančio elemento.
  • Position (top/bottom/left/right)
Box Model

Content

Elemento turinys tai ekrane atvaizduojama informacija vartotojui. gali būti tekstas, paveiksliukas, fonas ir pnš... Turinio dydį nusako turinio kiekis / dydis (nustatytas per CSS) / display.

Padding

Pamušalas - Sukurią gražius atitraukimus. Tai yra atstumas tarp elemento turinio ir rėmelių, kuris dažniausiai naudojamas tam, kad elemento turinys (content) nebūtų prilipęs prie savo kraštų.

  • padding: top right bottom left;
  • padding: top right/left bottom;
  • padding: top/bottom right/left;
  • padding: top/right/bottom/left;
  • padding-top: top;
  • padding-right: right;
  • padding-bottom: bottom;
  • padding-left: left;

Border

Rėmelis yra tarp padding ir margin. Rėmelis yra paskutinė atvaizuojama element dalis (fonas). Kitaip pasakius - rėmelis yra matomo elemento kraštas.

CSS rašymo būdai

  • Border: top/right/bottom/left;
  • Border-top: width type color;
  • Border-right: width type color;
  • Border-bottom: width type color;
  • Border-left: width type color;
  • border-width: top/right/bottom/left;
  • border-style: top/right/bottom/left;
  • border-color: top/right/bottom/left;
  • border-radius: topLeft I topRight I bottom/right I bottomLeft;
  • Border-right-color: rightColor;
  • border-[kraštinė]-[nustatymas]: nutstatymoParinktis

Margin

Paraštė yra atstumas nuo elemento rėmelio iki aplinkinių elementų, kuris dažniausiai naudojamas tam, kad elementą patrauktume nuo kitų elementų.
Margin negali įgauti FONO.

CSS rašymo būdai:

  • margin: top right bottom left;
  • margin: top right/left bottom;
  • margin: top/bottom right/left;
  • margin: top/right/bottom/left;
  • margin:-top: top;
  • margin:-right: right;
  • margin:-bottom: bottom;
  • margin:-left: left;
