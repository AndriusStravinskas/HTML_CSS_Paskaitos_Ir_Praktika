Media Queries

Media Queries - Naudojami kai norime, kad ekranui esant specifiniam pločiui būtų naudojami specifiniai stiliai (maži ekranai, turi kitokius stilius, nei dideli ekranai).

@media extra (min/max-width: 999px) {styles}

Galiima nurodyti, kad tam tikri stiliai galiotų tik tam tikrose ekrano pločio ribose:

@media extra (min-width: 500px) (max-width: 600px) { .... }

Breakpoints

Breakpoints - Ekrano lūžio taškai nurodytuose ekrano pločiuose. lūžio taškų pločiuos nusirodome kokius tik norime. Minimalistiškai skirtoma į 3 ekranus:

  • Mobile
  • Tablet
  • Desktop

Pvz.:

  1. 560px
  2. 760px
  3. 990px
  4. 1192px
  5. 1400px
